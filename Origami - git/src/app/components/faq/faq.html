Perguntas Frequentes

1. O Design System substitui a criatividade?

Não. Ele garante consistência, enquanto a criatividade resolve problemas reais de UX.

2. Posso criar novos componentes por conta própria?

Não. Qualquer novo componente deve passar pelo time do DS.

3. Quem aprova mudanças?

No time responsável pelo Design System (Design + Dev + Produto).

4. Por que devo seguir tokens e padrões?

Para garantir consistência, acessibilidade, economia de tempo e qualidade.

5. Como solicitar melhorias?

Envie um ticket com contexto, imagens, exemplos e justificativa.
