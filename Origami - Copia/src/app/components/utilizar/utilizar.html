Utilização

Antes de começar um projeto:

  • Verifique se já existe um componente para sua necessidade
  • Consulte o Figma e o Storybook
  • Verifique guidelines de comportamento
  • Siga tokens oficiais (cores, espaçamentos, tipografia)

Ao criar uma nova tela:

  • Utilize componentes existentes
  • Combine exatamente como documentado
  • Evite variações ou “gambiarras visuais”
  • Garanta acessibilidade mínima (contraste, foco)

Quando um componente não for suficiente?

  • Abra uma solicitação
  • Explique o contexto de uso
  • Envie protótipo ou caso real
  • Aguarde avaliação do time do DS

Fluxo simplificado

Buscar → Validar → Implementar → Contribuir → Evoluir
